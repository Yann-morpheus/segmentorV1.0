
              manuel d'utilsation de segmentor
========================================================================

  1> si vos fichier doivent avoir une entete ou un pieds de page fixe 

	-saisissez l'entete dans le fichier <header.txt>.
	-saisissez le pieds de page  dans le fichier <footer.txt>.

  2> si vos fichier n'ont pas une entete ou un pieds de page fixe  vous pouver laisser ces fichiers vides.
  3> deplacer le fichier a segmenter dans le repertoire parent ou se situe le l'executable de segmentor.
  4> lancer l'executable puis suiver les instructions...patienter quelques seconde  et vos segmentation sont pretes.

	    regles d'utilisation 
  ====================================================
  1> ne pas supprimer les fichiers header.txt et footer.txt.
